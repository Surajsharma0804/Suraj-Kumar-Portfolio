# Add Agentic Workflows Dashboard Image

## Quick Steps

1. **Save your dashboard screenshot** from the browser
   - Right-click on the dashboard page
   - Select "Save image as..." or take a screenshot
   - Save it as `project-agentic-workflows.png`

2. **Add to assets folder**
   - Place the image in: `src/assets/project-agentic-workflows.png`

3. **Update the import** in `src/components/ProjectsSection.tsx`
   - Uncomment line 6: `import projectAgenticWorkflows from "@/assets/project-agentic-workflows.png";`
   - Change line 12 from: `image: projectTaskManager,`
   - To: `image: projectAgenticWorkflows,`

- **File Size**: Under 500KB for fast loading
- **Content**: Your dashboard showing:
  - "Welcome to Agentic Workflows" header

  - Workflow Activity chart
  - Performance Metrics

## Alternative: Use Screenshot Tool

If you want a clean screenshot:

1. Open: https://agentic-workflows-pm7n.onrender.com
2. Press `F12` to open DevTools
3. Press `Ctrl+Shift+P` (Windows) or `Cmd+Shift+P` (Mac)
4. Type "screenshot" and select "Capture full size screenshot"
5. Save as `project-agentic-workflows.png`
6. Move to `src/assets/` folder
