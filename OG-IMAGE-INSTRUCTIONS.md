# Open Graph Image Instructions

## What You Need

Create an image file named `og-image.png` and place it in the `public/` folder.

## Image Specifications

- **Dimensions**: 1200 x 630 pixels (recommended for Open Graph)
- **Format**: PNG or JPG
- **File Size**: Keep under 1MB for fast loading
- **File Name**: `og-image.png`
- **Location**: `public/og-image.png`

## Design Suggestions

Your OG image should include:

1. **Your Name**: "Suraj Kumar" in large, bold text
2. **Title**: "Full Stack Developer & AI/ML Enthusiast"
3. **Background**: Use your portfolio's color scheme (dark background with teal/purple gradients)
4. **Optional Elements**:
   - Your profile photo
   - Tech stack icons (React, Python, JavaScript, etc.)
   - Your domain: "surajkumar.site"

## Quick Creation Options

### Option 1: Use Canva (Easiest)
1. Go to canva.com
2. Search for "Open Graph" template (1200 x 630)
3. Customize with your info
4. Download as PNG

### Option 2: Use Figma
1. Create a 1200 x 630 frame
2. Design your image
3. Export as PNG

### Option 3: Use Online Tools
- https://www.opengraph.xyz/
- https://ogimage.gallery/
- https://og-playground.vercel.app/

## After Creating

1. Save the image as `og-image.png`
2. Place it in the `public/` folder
3. The meta tags are already configured to use it
4. Test your OG image at: https://www.opengraph.xyz/url/https://surajkumar.site
